# Stellarity 末地

![](https://i.imgur.com/p7eHI03.png)

[Stellarity 的 Modrinth 页面](https://modrinth.com/datapack/stellarity)

本页面所有内容由 1Dot 手动格式化自 [Stellarity - kohara's Basement Wiki](https://koharasbasement.wiki.gg/wiki/Stellarity)，将所有条目展平并提供了与资源包一致的简体中文本地化对照，希望能提供更方便的阅读和搜索体验。

Stellarity 是一个数据包，它对末地进行了大量改造，赋予它新的地形形状、生物群落、物品和武器，甚至新的机制。Stellarity 并没有遵循 Minecraft 地下城风格扩展的常见主题或使其成为外星维度，而是将荒凉维度的常见主题与强烈的魔幻主题融合在一起。

> 有许多作者没有设立页面的 Stellarity 物品，这里也无法索引，请自己摸索发现
>

## 附魔

### <img alt="Invicon Netherite Sword.png: Sprite image for Netherite Sword in Minecraft" src="https://koharasbasement.wiki.gg/images/8/81/Invicon_Netherite_Sword.png?58ae4b" width="32" style="image-rendering: pixelated; display: inline-block; vertical-align: middle;"> 暴击之刃 Critical Strike

| 属性         | 值                       |
| :----------- | :----------------------- |
| **推荐物品** | 剑                       |
| **适用物品** | 剑，斧                   |
| **来源**     | 末地城的宝库战利品       |
| **不兼容**   | 锋利，亡灵杀手，节肢杀手 |
| **最高等级** | 3                        |

使攻击有 `附魔等级 * 10%` 的几率造成双倍伤害。

### <img alt="Invicon Arrow of Slow Falling.png: Sprite image for Arrow of Slow Falling in Minecraft" src="https://koharasbasement.wiki.gg/images/a/a0/Invicon_Arrow_of_Slow_Falling.png?6d099b" width="32" style="image-rendering: pixelated; display: inline-block; vertical-align: middle;"> 漂浮之箭 Levitation Shot

| 属性         | 值               |
| :----------- | :--------------- |
| **适用物品** | 弓               |
| **来源**     | 村民出售，权重 2 |
| **最高等级** | 5                |

使弓有 `[15 + 10 * (附魔等级 - 1)] / [100 + 12 (附魔等级 - 1)] * 100%` 的几率将箭矢变为悬浮之箭，对命中的目标施加悬浮效果。

效果的持续时间是随机的，并随等级提升。

- 最短时间为 4 秒。
- 最长时间为 7 秒。
  每一级都会使最短和最长持续时间增加 0.5 秒。

### <img alt="Invicon Elytra.png: Sprite image for Elytra in Minecraft" src="https://koharasbasement.wiki.gg/images/f/fd/Invicon_Elytra.png?48dfb8" width="32" style="image-rendering: pixelated; display: inline-block; vertical-align: middle;"> 装甲 Plated

| 属性         | 值               |
| :----------- | :--------------- |
| **适用物品** | 鞘翅             |
| **来源**     | 村民出售，权重 2 |
| **不兼容**   | 翱翔 (Soaring)   |
| **最高等级** | 4                |

增强鞘翅，使其提供 `1 + 附魔等级` 点护甲。

从等级 3 及以上开始，它还会提供 1 点盔甲韧性 (不受等级影响)。

每一级都会使飞行距离略微减少 `(1 + 附魔等级 * 3)%`。
